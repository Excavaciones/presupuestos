
# Generador de Presupuestos — Excavaciones y Ser. Arturo 

Herramienta web para generar presupuestos de obra de demolición y movimiento de tierras. Funciona directamente en el navegador, sin servidor ni instalación.

## 🚀 Acceso

👉

## ✅ Funcionalidades

- Generación de presupuestos con numeración automática anual
- Base de precios con más de 80 partidas organizadas en 9 categorías
- Buscador en la base de precios por descripción o código
- Datos de empresa y logo guardados en el navegador (localStorage)
- Exportación a Excel (.xlsx)
- Vista de impresión / guardado en PDF
- Firma de empresa y aceptación de cliente en el documento impreso

## 📂 Categorías de obra

| Categoría | Descripción |
|---|---|
| Pavimentos | Hormigón, adoquín, baldosas, soleras |
| Estructura | Pilares, cerchas, correas, cimentación |
| Cubiertas | Teja cerámica, zinc, panel sándwich |
| Fachadas | Obra vista, ladrillo, tabiquería, falso techo |
| Carpintería | Puertas, ventanas, barandillas |
| Instalaciones | Eléctrica, fontanería, fotovoltaica, saneamiento |
| Muros y Contención | Hormigón armado, piedra, ladrillo visto |
| Mov. Tierras | Excavaciones, rellenos, zahorras, desbroce |
| Auxiliares | Medios auxiliares, residuos, S&S, costes indirectos |

## 🛠️ Tecnología

- HTML5 + CSS3 + JavaScript puro (sin frameworks)
- [SheetJS](https://sheetjs.com/) para exportación a Excel
- Google Fonts (DM Sans, DM Mono)
- Sin base de datos — todo funciona en local con `localStorage`

## 📋 Uso

1. Abre la herramienta en el navegador
2. En **Datos y Logo Empresa**, rellena los datos de tu empresa y guárdalos
3. En **Nuevo presupuesto**, introduce cliente, dirección y fecha
4. Selecciona categoría y partida, ajusta la cantidad (casilla naranja)
5. Pulsa **VER PRESUPUESTO PARA IMPRIMIR** para generar el documento
6. Imprime o guarda como PDF desde el navegador
